# Como exportar os diagramas para PDF/DOC

## Opção 1: Mermaid Live Editor (mais fácil)

1. Acesse **https://mermaid.live**
2. Copie o conteúdo de `diagrama-classes.mmd` ou `diagrama-casos-uso.mmd`
3. Cole no editor à esquerda
4. Clique em **Actions** → **Download PNG** ou **Download SVG**
5. Insira a imagem no Word/Google Docs e exporte como PDF

## Opção 2: VS Code

1. Instale a extensão **Mermaid** ou **Markdown Preview Mermaid Support**
2. Abra o arquivo `.mmd` e use o preview (Ctrl+Shift+V)
3. Clique com o botão direito na imagem → Salvar como PNG

## Opção 3: HTML (imprimir como PDF)

1. Abra `diagramas.html` no navegador
2. Pressione Ctrl+P → Salvar como PDF

## Arquivos

| Arquivo | Descrição |
|---------|-----------|
| `diagrama-classes.mmd` | Diagrama de classes (Pet, Veterinário, Consulta, Pagamento) |
| `diagrama-casos-uso.mmd` | Casos de uso completos (todos os CRUD) |
| `diagrama-casos-uso-simples.mmd` | Casos de uso simplificados (fluxo principal) |

Use a versão que o professor preferir (simples ou completa).
